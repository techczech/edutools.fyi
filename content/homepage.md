## Four type of tools

This website illustrates four kinds of tools you can build:

1. **Interactive learning objects** - exercises, explainers, and more
2. **Content distribution** - readings, manuals, textbooks
3. **Agent skills** - operations manuals agents can reuse when you delegate work to them
4. **Desktop applications** - traditional applications you can install on a machine.

These describe what a thing is for. They do not rank projects by technical complexity or maturity.

## Three ways AI is involved
AI can be  involved in three different ways:
1. **Built by AI**  means that a coding agent built the tool as traditional software. The user is using a tool just like before but it was built by AI
2.  **Powered/augmented by AI** means that the traditional software has some AI component (for example an optional chat)
3.  **Delegated to AI**  means that this not traditional software but rather a set of instructions (SOPs) the agent can follow to do something on your behalf.


## Tools for building with AI
There are two kinds of tools that can help you build with AI:

1. **Hosted web builders:**  Tools like Google AI Studio or Lovable that will let you build and deploy learning experiences all inside a browser.
2.  **Desktop agents:** Tools like ChatGPT Work/Codex or Claude Code that run on your desktop and can build even desktop or phone apps.  These are also tools you can delegate tasks to using agent skills.
